O que é o HTML ?

(voltar)

HTML não é uma linguagem de programação; é uma linguagem de marcação, usada para definir a estrutura do seu conteúdo. HTML consiste de uma série de elementos, que você usa para delimitar ou agrupar diferentes partes do conteúdo para que ele apareça ou atue de determinada maneira.

Como o HTML é usado ?

(voltar)

HTML (Linguagem de Marcação de Hipertexto) é o código que você usa para estruturar uma página web e seu conteúdo. Por exemplo, o conteúdo pode ser estruturado em parágrafos, em uma lista com marcadores ou usando imagens e tabelas. Como o título sugere, este artigo fornecerá uma compreensão básica do HTML e suas funções

O que é Tag ?

(voltar)

Em HTML a tag é usada para criar um elemento. O nome de um elemento HTML é o nome usado entre colchetes angulares como "< j >". Note que ao fechar uma tag ela tem uma barra antes de seu nome, <>"< / j>" , e que em elementos vazios a tag final não é requirida e nem permitida. Se os atributos não forem mencionados, os valores padrões serão usados em cada caso.

Um exemplo de tag é a tag < h > usada para titulo, vai h1 ao h6 :

  1. Título

  2. Título

  3. Título

  4. Título

  5. Título
  6. Título
